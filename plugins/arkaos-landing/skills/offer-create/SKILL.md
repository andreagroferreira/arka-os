# Grand Slam Offer Creation

> **Agent:** Ines (Conversion Strategist) | **Framework:** Grand Slam Offer (Alex Hormozi)

## Value Equation

```
Value = (Dream Outcome x Perceived Likelihood of Achievement)
        / (Time Delay x Effort & Sacrifice)

Maximize: Dream Outcome, Perceived Likelihood
Minimize: Time Delay, Effort
```

## Offer Creation Steps

### Step 1: Dream Outcome
- What does the customer REALLY want? (not the product, the outcome)
- Frame as transformation: "From [current state] to [desired state]"

### Step 2: List ALL Problems
- Between the customer and their dream outcome, what problems exist?
- List 10-20 problems (obvious and hidden)
- For each problem, create a solution

### Step 3: Bundle Solutions
- Package solutions into a coherent offer
- Give it a unique, ownable name
- Structure: Core Offer + Bonuses + Guarantee

### Step 4: Bonus Stack
- 3-5 bonuses that address specific problems
- Each bonus has its own perceived value
- Stack value should be 5-10x the price
- Use "fast action" bonus for urgency

### Step 5: Guarantee
| Type | Risk Level | Example |
|------|-----------|---------|
| Unconditional | Lowest (for seller) | 30-day money back, no questions |
| Conditional | Medium | "If you don't get X result in Y days, full refund" |
| Anti-guarantee | Highest risk taken by buyer | "All sales final" (for premium) |

### Step 6: Pricing
- Price anchored by total value of stack
- "Total value: $15,000. Your investment: $997."
- Never compete on price. Compete on value.

### Step 7: Scarcity & Urgency (Real, Not Fake)
- Limited spots (if genuinely limited)
- Deadline (if genuine: launch window, cohort start)
- Bonus expiration (fast action bonuses remove after X days)
